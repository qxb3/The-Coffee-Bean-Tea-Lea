# Development

# Clone it
```sh
git clone https://github.com/qxb3/The-Coffee-Bean-Tea-Lea.git
cd The-Coffee-Bean-Tea-Lea
npm install
```

## Dev
```sh
npm run dev
```

## Building
```sh
npm run build
```
